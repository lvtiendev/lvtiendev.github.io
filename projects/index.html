AppMan

AppMan is one of the projects during my summer internship at Viki. It is a secured web-interfaced CMS running on the existing database. AppMan is in charge of managing all applications using Viki APIs. It helps the developers to register, manage and gain accesses to Viki services for their own applications. Moreover, it brings an easy way for Viki staffs to manage both internal and external applications across the system.
The website can be found here: https://appman.viki.io

Tech: Ruby on Rails

Streamify

Streamify is an iOS audio live broadcasting application. It allows users to host their own radio shows and choose to broadcast them to either friends or public audience. Using the app, users can stream the music directly from their iPads library, overlay with their voice and add some special sound effects to the stream. Listeners can tune in any channel to enjoy the shows and interact with the DJs through chat room. Everything happens in near real-time.

Tech: iOS PHP HTTP Live Streaming

CookieJar

Cookie Jar is a ToDo Manager, which helps users keep track of the upcoming events/school deadlines/tasks. The product’s audience is savvy computer users who prefer working only with keyboard. It supports flexible input commands from the user. For example, users can type “add having a meeting next Friday afternoon” into the command bar; then the program will understand and add the correct event into its database. Other functionalities like smart searching or editing an event are also supported.

Tech: Java
